my custom ubuntu vm
================

Download:
```
curl -L -o /tmp/install.sh https://raw.github.com/bySabi/my_custom_ubuntu_vm/master/install.sh
```

Install on Server
```
chmod +x /tmp/install.sh && sudo /tmp/install.sh
```

Install on Desktop
```
chmod +x /tmp/install.sh && sudo /tmp/install.sh desktop
```
